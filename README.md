# XV Engine
This project's first purpose is to make me learn the most of graphics programming in DX11 and OpenGL and C++ game good practices, techniques and stuff

## Build requirements
Inside "third_party"
```
    - fmt
    - GLAD (GL ver. 330)
    - GLM
    - KHR 
    - SDL2
    - stb
```

Have directx11 installed (already the case for most Windows users)
